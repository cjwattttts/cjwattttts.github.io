State Medical Policies Comparison

This project provides a comparative analysis of state medical policies, allowing users to input two states of their choice. The sheet returns rows containing data relating to the medical policies of each state. Green cells indicate policies that are supported in the state, and users can easily compare the support each state provides. Additionally, the sheet calculates and displays the total policy score for each state, providing a clear view of which states are more supportive of medical policies. A pie chart also visualizes the comparison, enhancing the overall user experience. The project includes multiple pivot tables and filters to allow users to analyze the data in-depth.

 

Tools Used: Excel

 

Formulas, Skills, and Techniques Used: VLOOKUP, IFERROR, SUM, Conditional Formatting, Pivot Tables, Pie Charts, Data Visualization
