Retail Performance & Trend Analysis

This project involved analyzing Western US sales trends, creating an interactive dashboard with visualizations and drill-down insights to identify outliers and key trends.

Tools Used:

  • Excel
  • Tableau

Project Overview:

During this project, I leveraged Excel for data cleaning and analysis, and Tableau to create interactive dashboards that allowed for in-depth analysis of sales performance across regions. The key objective was to identify outliers in the data and uncover trends that could inform business decisions.

Key Features:

  • Interactive profit maps
  • Drill-down functionality to explore specific regions
  • Dynamic charts for data visualization
