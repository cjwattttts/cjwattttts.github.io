Amazon Sales Analytics Case Study

For this self-directed case study, I analyzed a dataset of Amazon product listings using SQL (Snowflake), Excel, and Power BI. I wrote custom queries to calculate review volume, discount trends, and average product ratings. I then cleaned and joined the data to create meaningful metrics. In Power BI, I designed a polished dashboard with KPIs, pie and bar charts, and scatter plots to explore discount-to-review relationships, category breakdowns, and product trends. This project demonstrates my ability to work across the full data pipeline, from raw data to SQL transformation to clear, visual storytelling.

 

Tools Used: Snowflake (SQL), Excel, Power BI

 

Skills & Techniques: SQL Aggregations, Data Cleaning, Joins, KPIs, Visual Design, Trend Analysis, Dashboard Layout, Custom Formatting
